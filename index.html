HTML

  • The head element contains information about the webpage
  • The body element is what content the user will see
  • Dont use headings one through 6 to bold stuff
  • Unordered lists are like grocery store lists, sequence is not needed
  • Ordered lists are like directions, sequence is needed

CSS

  • A margin indicates how much space we want around the outside of an element
  • A padding indicates how much space we want around the content inside an element
  • You can set color by either typing out the word itself, or by using hexadecimal color code
  • Class selectors assign rules to a class attribute, rules will apply to everything with said class attribute
  • Dont forget to actually attach a link to the CSS stylesheet in the HTML header

Git

  • MKDIR means make a new directory at current location
  • PWD displays current working directory, where am I?
  • CD means change directory
  • Refer to Git cheat sheet I have made
